ECO2510 Data Project (with Xin Wang)


• ) The write-up :
PDF File: report.pdf
Word Document: report.docx

• ) R scripts:
merge.txt : to seperate the trade and quote files for individual firm and merge the files for the same firm on different months 
grid.txt : to generate the time grid and consolidate the trade data into the grid
events.txt : to identify events and compute estimates for reaction times using MAXMIN and NLS methods
regress.txt : to generate regression tables in the report
other.txt : to generate diagrams to illustrate the methods  with simulated data

(To run the code, replace the "path" variable at the beginning of each text file, set up two folders with names of the years, e.g. 2006 and 20111, and put the "trade_#month.csv" and "quote_#month.csv" files in the folders where #month is the string that is consistent with the "months" variable at the beginning of each text file.)

• ) Outputs:
2006 events.csv : the variables for 2006 events
2011 events.csv : the variables for 2011 events
regression.csv : the regression for the reaction time onto price differences, quantity traded, indicators of good news and years dummy.
