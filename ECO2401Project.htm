ECO2401 Term Project


• ) The whole project in a zip file: Dropbox link

• ) The write-up : ECO2401_Term_Project_Young_Wu.pdf

• ) Fortran codes: 
estimation.f90 : Estimation of parameter for coordination games ;
test.f90 : Generate test statistic for test of equilibrium belief ; 
power.f90 : Generate power of the test as a function of the parameters ; 
logit.f90 : Useful functions for Logit MLE, matrix inversion, element-wise matrix operation, etc

• ) R script: R_script.txt : to generate the plots given the output files

• ) Outputs:
u0_size_power.txt : Power of the test as a function of u0 (column 1 - value of u0 ; column 2 - size of test ; column 3 - power of test)
utilda_size_power.txt : Power of the test as a function of utilda (column 1 - value of utilda ; column 2 - size of test ; column 3 - power of test)
theta_size_power.txt : Power of the test as a function of theta (column 1 - value of theta ; column 2 - size of test ; column 3 - power of test)

chi_sq.txt : Chi-squared test statistic (column 1 - with equilibrium belief ; column 2 - with beliefs not in equilibrium)

estbelief_er.txt : Estimation of beliefs imposing ERs at every Z
estbelief_noer.txt : Estimation of beliefs imposing ERs only at the extremes
estparam_er.txt : Estimation of parameters imposing ERs at every Z
estparam_noer.txt : Estimation of parameters imposing ERs only at the extremes
